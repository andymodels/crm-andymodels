AndyRadio — front estático do andymodels.com (sem backend no domínio do site)
================================================================================

O player da página Rádio do site deve obter o JSON a partir do MESMO backend do CRM,
por exemplo:

  https://SEU-CRM.onrender.com/api/public/radio/v2

Substitua SEU-CRM pelo valor de PUBLIC_APP_URL (sem barra final).

No código do site (fetch / axios), use essa URL absoluta ou uma variável de ambiente
do build, por exemplo:

  NEXT_PUBLIC_RADIO_JSON_URL=https://....onrender.com/api/public/radio/v2

A resposta inclui os campos "crm_public_base" e "radio_json_url" para confirmar a origem.

Não é necessário outro Express/API só para a rádio no andymodels.com.

Desativar remoção automática de playlists de teste na 1.ª subida do servidor CRM:

  RADIO_SKIP_PURGE_TEST_ON_BOOT=1
